# LeaveFlow - Leave Management System

Initial project setup for the LeaveFlow Scrum group project.

## Planned Tech Stack

- Spring Boot backend
- React/Vite frontend
- MySQL database
- Flyway migrations
- JWT authentication
- AWS S3 document uploads
- AWS EC2 deployment

## Planned Features

- User authentication
- Role-based access control: ADMIN and EMPLOYEE
- Employee leave request creation
- Admin leave approval/rejection
- Leave history
- Supporting document uploads
- Dashboard summaries

## Project Structure

```text
backend/   Spring Boot API
frontend/  React/Vite frontend
docs/      Scrum and deployment documentation
```

## Collaboration Rules

- No direct commits to `main`
- Use feature branches
- Open Pull Requests
- At least one reviewer per PR
